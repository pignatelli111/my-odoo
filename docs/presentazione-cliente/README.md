## Ogni nuova foto (automatico via chat)

1. Invii screenshot + (opzionale) «Add to client report».
2. In chat ricevi in **inglese**: what it is, what to do, what to say.
3. Nel repo: nuova **Scheda** in `REPORT_PRESENTAZIONE_CLIENTE_IT.md` + file in `screenshots/`.
