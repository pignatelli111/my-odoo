C) Document & reference naming conventions (write your rules — examples)
------------------------------------------------------------------------
Use a single pattern so exports, folders, and Odoo `origin` fields stay readable.

Suggested patterns (edit to taste):

  **Projects / jobs**  
    `COMM-{year}-{seq}` or `{ClienteAbbrev}-{CantiereAbbrev}-{REV}`  
    Example: `COMM-2026-0042` or `ACME-CORNATE-REV01`

  **SBU estimates (custom module)**  
    Sequence already in Odoo; align verbal name with: `PRV-{number}-{REV}`

  **Purchase requests / RDA**  
    `RDA-{project}-{seq}` — match Excel if possible.

  **SAL / billing**  
    `SAL-{project}-{period}-{n}` e.g. `SAL-COMM-2026-0042-2026-05-01`

  **Chatter attachments (generic)**  
    Prefix: `{DocType}_{Ref}_{YYYYMMDD}_{shortdesc}.pdf`  
    Example: `RDA_COMM20260042_20260513_profili.pdf`

  **OneDrive folders (human)**  
    Same root as today; subfolder = project code above.
